# 201507_jjug_night_seminar
2015年7月のJJUGナイトセミナーに向けた資料作成リポジトリです。

- highandlow / buckets
登壇時に表示させるいわゆる「ダメコード」良いソースへの変換前ソース

登壇時最終的なソースコードとして表示させたい「回答コード」

java -classpath target/201507_jjug-1.0.jar highandlow_answer.HighAndLow
